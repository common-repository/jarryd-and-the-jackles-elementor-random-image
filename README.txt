== Description ==

A random collection of elementor widgets to help make adding obscure functionality easy.

== Installation ==

1. Upload the contents of `jarrydandthejackles-elementor-random-image.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

If using the popup builder (you need to do that in order to register the popup post types.):
- Go to settings > permalinks and click save.
